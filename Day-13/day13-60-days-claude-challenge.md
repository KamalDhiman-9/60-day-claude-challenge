# 🚀 60 Days of Claude Challenge — Day 13/60

## 🎯 Today's Focus: AI-Powered Job Search Assistant

Today I built a **job discovery & analysis workflow** using Claude, combining a structured profile + job search criteria system with live job search via the Indeed connector.

### 🛠️ What I Did
- Defined a reusable **Professional Profile** prompt (role, skills, experience, achievements)
- Defined a **Job Search Criteria** prompt (target roles, location, salary, exclusions)
- Used Claude to **search live internship listings** (Data Science / AI-ML roles across Remote + India-based hubs like Hyderabad, Noida, Pune)
- Got Claude to auto-generate a **ranked opportunity table** with:
  - Match scores
  - "Why it fits" reasoning
  - Skill gap analysis
  - Market demand insights
  - Actionable recommendations to boost interview chances

### 💡 Key Learnings
- Prompt chaining (Profile → Criteria → Discovery) makes multi-step AI workflows way more reliable than one giant prompt
- Claude's connector tools (like Indeed) turn it from a "chatbot" into a real **research + action assistant**
- Skill-gap analysis surfaced a clear gap for me: **GenAI/LLM exposure** (RAG, LangChain) is now expected even in entry-level AI/ML internships — adding this to my next project

### 📌 Next Steps
- Build a small RAG-based project to close the skill gap identified today
- Deploy 1-2 of my existing ML projects with a live demo link
